📌 Proje Hakkında - TR 
- Bu proje, Flutter framework’ü kullanılarak geliştirilmiş bir Türkçe–İngilizce kelime öğrenme uygulamasıdır.
- Uygulama, kelimeleri lokal veritabanında (Drift / SQLite) saklayarak çevrimdışı çalışabilmektedir.
- Kullanıcılar kelime ekleyebilir, silebilir ve kayıtlı kelimeleri listeleyebilir.
  
🚀 Özellikler
- 📱 Flutter ile geliştirilmiş modern UI
- 💾 Drift (SQLite) ile lokal veritabanı kullanımı
- ➕ Türkçe–İngilizce kelime ekleme
- 🗑️ Kelime silme
- 📃 Kayıtlı kelimeleri listeleme
- ⚡ Offline çalışma desteği
- 🧱 MVVM / Clean Architecture yapısına uygun geliştirme
  
🛠️ Kullanılan Teknolojiler
- Flutter
- Dart
- Drift (SQLite)
- Provider / State Management
- Material Design

- Projeyi klonlayın:
  git clone https://github.com/kullaniciadi/proje_adi.git
- Gerekli paketleri yükleyin:
  flutter pub get
- Drift için build işlemini çalıştırın:
  flutter pub run build_runner build
- Uygulamayı başlatın:
  flutter run

📌 About the Project - EN
- This project is a Turkish–English vocabulary learning application developed using Flutter.
- It stores words in a local database using Drift (SQLite), allowing the app to work offline.
- Users can add, delete, and view saved words easily.
  
🚀 Features
- 📱 Modern UI built with Flutter
- 💾 Local database with Drift (SQLite)
- ➕ Add Turkish–English words
- 🗑️ Delete words
- 📃 List saved vocabulary
- ⚡ Offline support
- 🧱 MVVM / Clean Architecture friendly structure
- 
🛠️ Technologies Used
- Flutter
- Dart
- Drift (SQLite)
- Provider / State Management
- Material Design
